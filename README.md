# Subgraph Arweave Example

## Setup

0. Get submodules

```
git submodule update --init
```

1. Setup environment

```
yarn setup
```

2. Run services

```
yarn up
```

## Build

```
yarn
yarn codegen
yarn build
```

## Create subgraph

```
yarn create-local
```

### Deploy subgraph

```
yarn deploy-local
```
